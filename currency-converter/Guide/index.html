A) How to Install - top

To get setup and ready to use Exchange Rate Converter, simply copy the currency_convert_class.php file from the ZIP archive you downloaded into a directory, of your choice - usually your "classes" folder, within your project.

B) Using Exchange Rate Converter - top

Exchange Rate Converter is designed to be easy and straight forward to use.

1) The first step is to include the Exchange Rate Converter class library. Make sure you specify the path to where the file is located.

include("path/to/currency_convert_class.php");

2) The next step is to create an object of the Exchange Rate Converter class:

$convertObj = new CurrencyConvert();

I've called the object $convertObj; you can call this what every you like.

3) In this step we do the conversion. We call the currencyConvert method, passing it the amount, the from currency, and the currency we want the result of.

The result is returned and saved to the variable $result.

$result = $convertObj -> currencyConvert($amount, $from, $to);
Params:
  • $amount: The amount you wish to convert
  • $from The currency you wish to convert
  • $to The currency you want the result of

4) In this final step, we display the result.

The result is returned as a floating, 2 decimal placed, value. To get the result in a different format, see the Displaying Formatted Results Examples section.

echo $result;

C) Displaying Formatted Results Examples - top

To give you control over how you wish to display the results, check out these full examples that show you what is possible.

All examples use an amount of $100 and convert from US Dollars to New Zealand Dollars.

Unspecified number of decimal places

To return the result as an unrounded value for maximum accuracy, call the getResult() method (highlighted below).

Code

include("currency_convert_class.php");
$convertObj = new CurrencyConvert();
$convertObj -> currencyConvert(100, 'USD', 'NZD');
echo $convertObj -> getResult();

Result

124.115676

Specified number of decimal places

To return the result as a rounded value with a specified number of decimal places, call the getResult() method, passing the desired number of decimal places.

In this example, we want out value to be returned to 3 decimal places

Code

include("currency_convert_class.php");
$convertObj = new CurrencyConvert();
$convertObj -> currencyConvert(100, 'USD', 'NZD');
echo $convertObj -> getResult(3);

Result (to 3dp)

124.116

Full text

If you wish to include the full name of the countries in the conversion, this is done by calling the getFullResult() method.

Code

include("currency_convert_class.php");
$convertObj = new CurrencyConvert();
$convertObj -> currencyConvert(100, 'USD', 'NZD');
echo $convertObj -> getFullResult();

Result

100 U.S. dollars = 124.115676 New Zealand dollars

Full text with parameters

getFullResult() can take two optional parameter like so:

getFullResult($number_of_dp, $use_country_code)

  • $number_of_dp (int): The number of decimal places to display. Default is unspecified number of decimal places. Accepts NULL
  • $use_country_code (boolean) Display the currency code (true), or display the country name (false). Default is country name.

Code

include("currency_convert_class.php");
$convertObj = new CurrencyConvert();
$convertObj -> currencyConvert(100, 'USD', 'NZD');
echo $convertObj -> getFullResult(2, true);

Result

100 USD = 124.12 NZD
